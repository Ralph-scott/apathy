# Apathy
apathy is the opposite of vim

## Controls
* `<Up>`/`<Down>`/`<Left>`/`<Right>`: move cursor
* `<Backspace>`: backspace
* `<Ctrl-c>`: save and exit
* `<Anything else lol>`: type that character in

## Usage
### Build
``` bash
./build.sh
```
### Run
``` bash
./apathy <file name>
```

## Software Requirements
* ncurses (for terminal graphics)
* gcc (to build executable)
